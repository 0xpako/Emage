### TL;DR

Emage framework uses Build Output API to support features such as:

- Static Rendering
- SSR
- Edge Rendering 
- Automatic Image Optimization

### How to use 

- Installing: `yarn install`
- Building: `yarn build:css && yarn build` 
- Deploying: `yarn deploy`
